## 📖 프로젝트 소개

## 💡주요 기능
### 1. 회원 관리 (담당 : 최사랑)
- 회원가입
- 로그인
- 로그아웃
- 정보 수정

### 2. 상품 관리 (담당 : 한인규, 송수미)
- 상품 등록 수정 삭제
- 상품 상세
- 상품 결제
- 상품 찜
- 거래 후기

### 3. 채팅 관리 (담당 : 강경민)
- 채팅
- 채팅 목록
- 채팅 상세

### 4. 관리자 페이지 (담당 : 강경민)
- 공지사항 등록 수정 삭제
- 회원 관리
- 상품 관리

### 5. 페이징 모듈 (담당 : 송수미)
