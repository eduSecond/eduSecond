## 프로젝트 소개
> 선생님을 위한 중고거래 사이트 당신마켓
- 선생님을 위한 당신마켓은 중고거래가 필요한 교사가 손쉽게 필요한 물품들을 중고거래 할 수 있도록 도와주는 사이트입니다.
